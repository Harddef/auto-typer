# Review Generator

Инструмент для генерации отзывов с помощью AI и поддержкой OCR для извлечения текста из изображений (чеки, визитки и т.д.).

[English](README.md) | **[Русский](README.ru.md)**

## Возможности

- 📸 **Загрузка изображений** - Загрузите до 5 изображений (чеки, визитки, документы)
- 📋 **Поддержка буфера обмена** - Нажмите Ctrl+V для вставки изображений из буфера
- 🔍 **OCR распознавание текста** - Автоматическое извлечение текста из изображений с помощью встроенного Windows OCR
- 🤖 **AI генерация отзывов** - Создание аутентичных отзывов с помощью Claude AI
- 🏪 **Типы услуг** - Поддержка ресторанов, магазинов, медицинских услуг, отелей и др.
- 🌍 **Русский язык** - Генерирует отзывы на русском языке с правильной грамматикой
- 🎨 **Современный тёмный интерфейс** - Чистый, профессиональный дизайн
- ⚙️ **Настраиваемость** - Настройки API через appsettings.json

## Требования

- Windows 10 (версия 1809 или новее) или Windows 11
- .NET 6.0 Runtime или новее
- Локальный AI API endpoint (OpenCode или совместимый Claude API)

## Установка

1. Скачайте последнюю версию со страницы [Releases](../../releases)
2. Распакуйте архив
3. Настройте `appsettings.json` с вашими настройками API
4. Запустите `ReviewGenerator.exe`

## Конфигурация

Отредактируйте `appsettings.json` для настройки API:

```json
{
  "ApiSettings": {
    "ApiKey": "ваш-api-ключ",
    "ApiEndpoint": "http://localhost:20128/v1",
    "Model": "kr/claude-sonnet-4.5"
  }
}
```

## Сборка из исходников

### Требования
- .NET 6.0 SDK или новее
- Visual Studio 2022 или JetBrains Rider (опционально)

### Инструкция по сборке

```bash
# Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/review-generator.git
cd review-generator/ReviewGenerator

# Соберите проект
dotnet build -c Release

# Или опубликуйте как единый исполняемый файл
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true -o publish
```

Исполняемый файл будет в `publish\ReviewGenerator.exe`

## Использование

1. **Загрузите изображения**: Нажмите "Add Images" или Ctrl+V для вставки из буфера
2. **Выберите тип услуги**: Выберите тип услуги (ресторан, магазин, медицина и т.д.)
3. **Добавьте инструкции** (опционально): Укажите дополнительный контекст или требования
4. **Сгенерируйте отзыв**: Нажмите кнопку "Generate Review"
5. **Просмотрите результаты**: 
   - Извлечённый OCR текст появится в средней секции
   - Сгенерированный отзыв появится внизу
6. **Скопируйте отзыв**: Нажмите "Copy" для копирования отзыва в буфер обмена

## Технические детали

- Создано на C# и Windows Forms
- Использует Windows.Media.Ocr (встроенный Windows OCR) для извлечения текста
- Поддержка множества языков на основе языковых настроек Windows
- Интеграция с Claude AI API для генерации отзывов
- Управление конфигурацией через Microsoft.Extensions.Configuration
- Современный тёмный интерфейс с компактным, чистым дизайном
- Полностью бесплатный и open-source без требований лицензии
