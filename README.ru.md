# Auto Typer

## Возможности

- 🎯 **Выбор целевого окна** - Кликните для выбора любого окна для ввода
- ⌨️ **Автоматический ввод** - Имитирует естественный ввод с клавиатуры через SendInput API
- ⚡ **Настраиваемая скорость** - Настройте задержку между символами (0-5000мс)
- 🌙 **Тёмная тема** - Современный, чистый интерфейс в тёмных тонах
- 🔥 **Поддержка горячих клавиш** - Нажмите F9 для быстрого старта/остановки
- 📝 **Поддержка многострочного текста** - Обрабатывает текст с переносами строк (Shift+Enter)
- 🌍 **Поддержка Unicode** - Работает с русским, английским и другими языками

## Требования

- Windows 7/8/10/11
- .NET 6.0 Runtime или новее

## Установка

1. Скачайте последнюю версию со страницы [Releases](../../releases)
2. Распакуйте `AutoTyper.exe` из архива
3. Запустите `AutoTyper.exe`

## Сборка из исходников

### Требования
- .NET 6.0 SDK или новее
- Visual Studio 2022 или JetBrains Rider (опционально)

### Инструкция по сборке

```bash
# Клонируйте репозиторий
git clone https://github.com/Harddef/auto-typer.git
cd auto-typer

# Соберите проект
dotnet build -c Release

# Или опубликуйте как единый исполняемый файл
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true -o publish
```

Исполняемый файл будет в `publish\AutoTyper.exe`

## Использование

1. **Введите текст**: Напишите или вставьте текст, который хотите автоматизировать, в текстовое поле
2. **Выберите целевое окно**: Нажмите кнопку "Select" и затем кликните на окно, куда должен вводиться текст
3. **Установите скорость ввода**: Настройте задержку между символами (по умолчанию: 50мс)
4. **Начните ввод**: Нажмите "Start Typing" или клавишу F9
5. **Остановка**: Нажмите "Stop" или снова F9 для отмены

### Важные замечания

- ⚠️ **Не кликайте на окно Auto Typer во время ввода** - это перенаправит ввод в само окно Auto Typer
- Окно будет оставаться поверх других во время ввода для отображения прогресса
- Целевое окно должно оставаться открытым во время ввода

## Горячие клавиши

- `F9` - Старт/Стоп ввода

## Технические детали

- Создано на C# и Windows Forms
- Использует Win32 `SendInput` API для симуляции клавиатуры
- Поддерживает Unicode символы через флаг `KEYEVENTF_UNICODE`
- Выбор окна через `WindowFromPoint` API
